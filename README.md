# Suporte do RDCRIA

Portal profissional de atendimento para streamings, com chat por protocolo, painel administrativo e controle de ponto da equipe.

## Acesso do painel
- Usuario: `Num`
- Senha: `rdcria2026`

Esses dados nao aparecem no site publico.

## Como iniciar
### Windows
Execute `iniciar.bat`

### Linux ou macOS
Execute:
```bash
chmod +x iniciar.sh
./iniciar.sh
```

### Manual
Execute:
```bash
node server.js
```

## Enderecos
- Site publico: `http://localhost:3000`
- Painel: `http://localhost:3000/admin`

## Recursos desta versao
- atendimento para streamings com abertura por protocolo
- chat entre cliente e painel
- controle de status dos servicos
- sistema de ponto da equipe com entrada, saida e historico
- responsivo para celular, tablet e computador
- sem credenciais exibidas no site publico
